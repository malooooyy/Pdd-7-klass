Будь на дороге, как дома

Простые и понятные правила дорожного движения для всех, кто ходит и ездит.

Главное для твоей безопасности

Правила для пешехода

  • Смотри по сторонам. Убери телефон и сними наушники, когда переходишь дорогу.
  • Используй "зебру". Переходи дорогу только по пешеходному переходу.
  • Жди зеленый. Даже если машин нет, иди только на зеленый свет светофора.
  • Будь заметным. Вечером носи одежду со светоотражателями.

На самокате и велосипеде

  • Спешивайся. Переходи дорогу по "зебре" только пешком, ведя транспорт рядом.
  • Один самокат — один человек. Кататься вдвоем на одном самокате запрещено.
  • Знай свою дорогу. Ездить можно по велодорожкам, а если их нет — по тротуару, не мешая пешеходам.
  • Надень шлем. Это не просто аксессуар, это твоя защита.

Проверь себя!
